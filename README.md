# GeoAILiteracy
This repository is used to share class resources for GGIS 427 GeoAI Literacy. To contribute a resource in [ResourceList.md](ResourceList.md), please include a clear citation of the resource. If you are submitting a link, then a working URL, and citation should be provided. If you are sharing a PDF you can either submit a link to the pdf's location on the web or the PDF itself.
